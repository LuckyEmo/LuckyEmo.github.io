这是一篇有关如何使用 Github Pages 和 Hexo 搭建属于自己独立博客的详尽教程，这也是我搭建好自己的博客之后写的第一篇博客，刚开始搭建博客的时候自己也是网上各种百度，由于自己属于小白那种，查阅了各种各样的文档，最后终于搭建好了自己的第一个博客，同时希望各路大佬在后面的博客优化时，给予我更多的建议。

安装node.js与hexo

访问官网，安装相应版本，然后无脑下一步即可.

  • node.js官网的安装地址：(https://nodejs.org/en/)
  • git官网的安装地址：(https://git-scm.com/)

安装完成后，进入命令提示符输入cmd后，输入一下命令，出现对应版本的信息则安装成功，反之则没有安装成功.

  • npm -v
  • node -v
  • git –version

安装hexo

  • 在磁盘中创建一个新的文件夹用来缓存hexo配置的一些文件，文件夹的命名可以随机命（我的文件名为LuckyEmo_Blog).
  • 当你创建好文件夹后，进入该文件夹的根目录，点击右键；你会发现会有Git Bash Here和Git GUI Here，则点击Git Bash Here.
  • 点击Git Bash Here后，输入以下npm命令：
    • npm install hexo-cli -g
    • npm install hexo-deployer-git –-save

hexo初始化

  • 首先还是进入你前面创建的文件夹的根目录，点击右键后，点击Git Bash Here

  • 输入hexo指令：hexo init

    ![](C:\Users\zhuan\Desktop\hexo init.png)

  • 如果hexo init出现报错target not empty
